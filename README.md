# CundiReferencias

## Descrpcion
CundiReferencias es una plataforma web desarrollada como proyecto de aprendizaje colaborativo. El proposito de la plataforma es proporcinar a los estudiantes un lugar donde puedan compartir y consultar opiniones, referencias y experiencias sobre los profesores de la universidad de Cundinamarca para apoyar la toma de deciciones a la hora de inscribir materias.

## Objetivo
El objetivo principal es fortalecer habilidades de desarrollo de software mediante la creacion de un sitio web real, aplicando buenas practicas de: trabajo en equipo, control de versiones, arquitectura de software, y desarrollo full stack.
Adicionalmente el proyecto sirve como espacio de aprendizaje para tecnologias y metodologias modernas y mas demandadas en la industria.
